# How to Run (DEMO DAY):
1. Launch the HTML file 'landing-page.html', as a live server.
2. On your MetaMask chrome extension, open the wallet of your desired stakeholder. 
3. Click connect wallet and click the confirm pop up on MetaMask. 
4. Select the correct role based on the stakeholder selected.
